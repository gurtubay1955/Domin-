# Arquitectura de Agentes y Plan de Implementación

Este proyecto está orquestado por **5 Agentes Especializados** cuyos prompts detallados se encuentran en esta carpeta:

1.  **[Supervisor Agent](./Supervisor_Agent.md)**: Coordinación general.
2.  **[Expert Rules Agent](./Expert_Rules_Agent.md)**: Reglas y arbitraje.
3.  **[Arquitecto de Torneos](./Arquitecto_de_Torneos.md)**: Logística y combinatoria.
4.  **[Backend Agent](./Backend_Agent.md)**: Lógica de servidor y Base de Datos.
5.  **[Front end Agent](./Frontend_Agent.md)**: Diseño UI/UX y experiencia visual.

---

## Flujo de Trabajo Actualizado
1.  **Supervisor** recibe requerimientos.
2.  **Expert Rules** valida cambios en el juego.
3.  **Backend** prepara la estructura de datos.
4.  **Frontend** implementa la interfaz visual pulida.
